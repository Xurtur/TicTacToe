# Tic Tac Toe

## About
This is a simple tictactoe game created using vite and ReactJS.<br>
This project is for personal learning and intended for portfoli purposes.

## Systems Integrated
- Player turns.
- Winner detection.
- Prevention of changing already selected tiles.
- Winner Announcement.

## Known Issues
- If game is a draw there is no way to reset the board; a hard refresh is required.
